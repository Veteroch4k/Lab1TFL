Прототип функции в языке PHP – это объявление сигнатуры функции, включающее её имя, список ожидаемых аргументов (с указанием типов и значений по умолчанию) и возвращаемый тип данных.
Для описания прототипа используется служебное слово "function".
Формат записи: "function имя_функции(тип $аргумент = значение, …): возвращаемый_тип;".
Примеры:
1.	Простая функция: function test(int $a);
2.	Функция с несколькими аргументами и возвращаемым типом: function calcTax(int $amount, string $currency = 'USD'): float;
3.	Функция без указания типов: function doSomething($var);
В связи с разработанной грамматикой G[Prototype], синтаксический анализатор (парсер) должен принимать на вход поток токенов от лексического анализатора и выносить решение о синтаксической корректности последовательности. При обнаружении ошибок анализатор должен нейтрализовать их и продолжить разбор для выявления последующих ошибок.
